Learning CSS

This is my first CSS practice file. :

  • pixels ---> px
  • ems ---> em
  • rems ---> rem
  • percentages ---> %
  • viewport width ---> vw
  • viewport height ---> vh
  • rem -> font size of the root element (html)
  • em -> font size of the parent element, if not defined, it will inherit from the root element (html)
  • percentage -> relative to the parent element, if not defined, it will inherit from the root element (html)
  • pixels -> absolute unit, fixed size, not responsive
  • vh -> 1% of the viewport height, responsive unit
  • vw -> 1% of the viewport width, responsive unit
